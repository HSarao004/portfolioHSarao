Project 1: WristRotation

Content Management System (CMS) Built with PHP and MySQL

This CMS is a dynamic and user-friendly platform designed for managing and publishing content. Developed using PHP and MySQL, it offers comprehensive features for both administrators and regular users.

Key Features

  • Admin Features:
    • CRUD Operations: Admins can create, read, update, and delete posts, giving them full control over the websiteâ€™s content.
    • Post Management: Admins can manage published posts, edit content, add media, and categorize posts for better organization.
  • User Features:
    • User Registration and Authentication: Users can register, log in, and securely manage their profiles.
    • Commenting System: Registered users can comment on posts, engaging in discussions and sharing their opinions on various topics.
    • Post Interaction: Users can view posts, search for content, and interact through comments.

Project Gallery

The index page where users land upon visiting the website, prompting them to log in or sign up.

Users have the option to sign up if they don't have an account. Their credentials are saved in the database with hashed passwords for security.

Users with existing accounts can log in, and the website verifies credentials with the database for secure access.

After logging in, users are welcomed and can view all watch posts. Three different filtering options are available for the posts.

Users are provided with pagination functionality for convenient browsing through posts.

Clicking on a watch name leads to a detailed post page with more information about the watch, along with comments from other users.

The admin dashboard allows admins to view and edit watch posts, with options to update or delete them.

The edit post page enables admins to modify any information about a watch post or delete it if necessary.

Admins can create, edit, or delete categories assigned to watches for better content management.

Admins have the option to create or delete user accounts, ensuring proper user management on the platform.
